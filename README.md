# CMB_WD_19 – WeatherApp Android

## Project Overview

WeatherApp is a simple Android mobile application developed for the Human Computer Interaction (HCI) Assignment 02.

The application provides weather information using external APIs and stores user-related data using a database.

## Module

Human Computer Interaction (HCI)

## Group

CMB_WD_19

## Technologies

* Java
* Android Studio
* Firebase / SQLite
* RecyclerView
* REST APIs

## Main Features

* View current weather
* Search weather by city
* Save weather data
* View weather history
* Responsive mobile UI

## APIs Used

1. Weather API
2. Location / Geocoding API

## Database

Firebase / SQLite
